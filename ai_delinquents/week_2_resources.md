These resources can used to better understand Github, Visual Studio Code (VS Code) as a tool for python coding, accessing Copilot through PNNL, best data cleaning practices, and what to include in creating strong data dictionaries.

## General Github documentation
Accessing copilot through VS Code (for example): https://docs.github.com/en/copilot/how-tos/get-code-suggestions/get-ide-code-suggestions?tool=vscode

In VS Code, if PNNL copilot has been used previously, can access by clicking on the conversation bubble square up top next to the search bar. 

## PNNL Forge 
PNNL Forge has disabled Github self service due to capacity limits (this is how we previously accessed copilot), but can access resources here, particularly if already have an active Github profile. Other questions can be directed to Research Computing: https://forge.pnnl.gov/home
